# SOBO STUDIO CMS - Quick Start

## 🎉 Your WordPress-Style Admin Panel is Ready!

You now have a fully functional content management system where you can edit all text and images on your SOBO STUDIO website without touching any code.

---

## 🚀 First Time Setup (5 Minutes)

### Step 1: Create Admin Account
1. Go to: **`/admin/signup`**
2. Fill in:
   - Your name
   - Email address (can be anything, no verification needed)
   - Password (minimum 6 characters)
3. Click **CREATE ACCOUNT**

### Step 2: Login
1. Go to: **`/admin/login`**
2. Enter your credentials
3. You're in!

### Step 3: Initialize Content
1. You'll see a welcome screen
2. Click **"Initialize Default Content"** button
3. Wait a few seconds
4. All content loads automatically!

### Step 4: Start Editing
1. Click on any of the 3 tabs: Hero, Services, or Work
2. Edit text in any field
3. Upload new images or paste image URLs
4. Click **SAVE ALL** when done

---

## 📍 Key URLs

- **Homepage**: `/`
- **Admin Login**: `/admin/login`
- **Admin Dashboard**: `/admin/dashboard`
- **Create Account**: `/admin/signup`

You can also click "Admin" in the navbar (top right) to access the login page.

---

## ✨ What You Can Edit

### Hero Section (Homepage Top)
- Main headlines (2 lines)
- Services text
- Tagline
- Hero image

### Services (3 Services)
- Service titles (Identity, Signal, Blueprint)
- Subtitles
- Descriptions
- Service images

### Work (4 Projects)
- Client names
- Categories
- Descriptions
- Project images

---

## 💾 How to Save Changes

**IMPORTANT**: Changes are NOT auto-saved!

1. Make all your edits across any tabs
2. Click the **"SAVE ALL"** button (top right)
3. Wait for "Content saved successfully!" toast
4. Changes are now live!

---

## 🖼️ Adding Images

**Two Ways:**

**Method 1: Upload**
- Click "UPLOAD" button
- Select image from computer
- Wait for upload to complete

**Method 2: Paste URL**
- Copy image URL from anywhere
- Paste into text field
- Image updates instantly

**Remove Image:**
- Hover over image preview
- Click X button

---

## 🔍 Preview Changes

Click **"PREVIEW SITE"** button to open your live site in a new tab and see your changes.

---

## 🔐 Logout

Click the logout icon (arrow) in the top right corner when you're done.

---

## 💡 Pro Tips

1. **Work in batches** - Make multiple edits before saving
2. **Preview often** - Check how changes look on the live site
3. **Keep images consistent** - Use similar aspect ratios
4. **Stay on brand** - Keep that bold, minimalist SOBO vibe
5. **Backup text** - Keep important copy in a separate doc
